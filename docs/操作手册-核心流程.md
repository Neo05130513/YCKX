# 延成科讯管理系统核心流程操作手册

本文档面向实施、培训和试运行人员，说明 OA 审批、租赁订单、系统管理和客户移动端的常用操作路径。

## 1. OA 审批

| 场景 | 操作路径 | 结果 |
| --- | --- | --- |
| 发起审批 | 后台进入 `OA审批`，点击 `新建审批`，选择模板，填写表单、附件和说明，提交审批 | 生成审批单并进入节点流转 |
| 处理待办 | 切换 `待我处理` 或点击 `待我审批` 队列，进入详情后通过、驳回、转办或备注 | 当前节点记录处理意见 |
| 补充附件 | 审批详情点击 `补充附件`，填写附件名称或选择本地附件 | 附件进入审批归档 |
| 批量处理 | 使用 `批量操作` 的催办、通过、转办或流程模板配置 | 快速处理同一筛选范围内的审批单 |
| 交付检查 | 审批详情查看 `交付检查清单` | 确认业务关联、节点、字段、附件、日志和时限 |

## 2. 租赁订单

| 场景 | 操作路径 | 结果 |
| --- | --- | --- |
| 新建订单 | `租赁订单` 点击 `新建订单`，填写客户、联系人、租期、数量、金额和合同名称 | 生成订单草稿并进入合同页 |
| 合同会签 | 订单详情点击 `发起会签`，会签完成后点击 `归档合同` | 订单可进入出库和履约 |
| 出库绑定 | 点击 `创建出库`，选择订单、仓库、收货人、地址和 BT 码 | 创建出库单，绑定电池并更新出库进度 |
| 确认收货 | 后台出库单点击确认，或客户移动端进入 `收货` 确认整张出库单 | 订单待收货数减少，设备进入租赁中 |
| 生成账单 | 订单详情 `账单回款` 点击生成下期账单，财务确认后登记回款 | 订单应收、已收和欠款同步 |
| 续租 | 订单详情点击 `续租`，填写新到期日、月租、押金和备注 | 合同状态更新，保留续租记录 |
| 退租 | 订单详情点击 `退租`，填写归还仓库和 BT 码，欠款结清后点击 `退租结清` | 资产回库，订单完成或进入结算 |

## 3. 系统管理

| 场景 | 操作路径 | 结果 |
| --- | --- | --- |
| 新增账号 | `系统管理` → `账号管理` → `新增账号`，选择内部或客户账号，绑定角色 | 用户可按角色登录 |
| 调整权限 | `角色权限` 中复制或配置角色，勾选权限并保存 | 菜单和数据权限更新 |
| 维护字典 | `状态字典` 选择分组，编辑显示名、业务规则和启用状态 | 业务状态口径统一 |
| 配置接口 | `接口配置` 编辑地址、鉴权、超时、负责人，保存后测试 | 第三方接口状态可验证 |
| 配置审批流 | `审批流程` 新增或编辑流程，设置节点和 SLA | OA 模板与系统配置保持一致 |
| 交付检查 | `总览` 点击 `一键交付检查` 或 `导出验收清单` | 输出上线前整改项 |
| 配置备份 | 点击 `导出配置` 保存 JSON 快照，必要时 `恢复配置` | 交付和回滚有依据 |

## 4. 客户移动端

| 场景 | 操作路径 | 结果 |
| --- | --- | --- |
| 查看待办 | 客户登录移动端首页，查看待收货、报修、账单/订单卡片 | 进入对应业务处理 |
| 查看订单 | 进入 `订单`，按全部、待收货、租赁中、退租中、已完成筛选 | 客户了解履约状态 |
| 确认收货 | 进入 `收货`，核对出库单、BT 码、自动收货日、收货地址，点击确认 | 后台出库单和订单状态同步 |
| 提交报修 | 进入 `报修`，填写故障描述、联系人、地址和图片/附件说明 | 后台生成售后工单 |
| 查看账单 | 客户管理员进入 `账单`，查看欠款、已付和逾期信息 | 可提交付款凭证 |
| 员工权限 | 客户员工进入系统时不展示金额 | 满足最小权限要求 |

## 5. 角色交接路径

| 角色 | 日常路径 |
| --- | --- |
| 销售 / 客户经理 | 新建订单 → 合同会签 → 合同归档 → 跟进客户收货 |
| 仓库 / 资产 | 固定资产库存 → 订单出库 → BT 码绑定 → 退租/维修入库 |
| 财务 | 账单确认 → 回款登记 → 催收队列 → 退款/调整审批 |
| 售后 | 客户报修 → 工单受理 → 维修跟进 → 待入库 → 客户确认 |
| 系统管理员 | 账号 → 角色 → 接口 → 审批流 → 参数 → 交付健康度 |
| 客户管理员 / 员工 | 首页待办 → 订单 → 收货 → 报修 → 账单或付款凭证 |

## 6. 试运行建议

1. 先用测试客户完整跑通订单创建、会签、出库、客户收货、账单、回款和退租。
2. 再用客户员工账号验证金额不可见、收货和报修可用。
3. 最后由系统管理员导出配置快照、系统验收清单和审计日志。
